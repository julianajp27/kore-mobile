# 🎓 KORE - Mobile

Aplicativo móvel desenvolvido para o Sistema de Gestão de Atividades Complementares. 

Este projeto permite que os alunos enviem seus certificados, acompanhem as horas aprovadas e gerenciem suas pendências diretamente pelo celular, integrando-se com a API do sistema web.

## 🚀 Tecnologias Utilizadas
* React Native
* Expo
* AsyncStorage (Armazenamento local)
* Integração com API (Fetch e FormData)
